# Whiskrs Sharing Board

[Whiskrs Sharing Board](https://.com/)

## Getting Started

Just clone, install dependencies, and run 'npm run dev' to concurrently run the server and client.

### Prerequisites

Node , express & react.

## Deployment

Fully deployed to heroku.

## Built with the MERN stack

- MongoDB
- Express
- React
- Node
